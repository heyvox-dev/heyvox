  <name>Task 1: Add HUD_SOCKET_PATH to constants and wire HUDClient into main.py</name>
  <files>vox/constants.py, vox/main.py</files>
  <action>
**In vox/constants.py:**

Add after the TTS_CMD_FILE constant:

```python
# HUD overlay Unix socket path (Phase 5)
# Used by HUDClient (sender in main.py/tts.py) and HUDServer (receiver in overlay.py)
# Requirement: HUD-08
HUD_SOCKET_PATH = "/tmp/vox-hud.sock"
```

**In vox/main.py:**

1. **Add import** at the top imports section:
   - Add `from vox.constants import HUD_SOCKET_PATH` (extend existing constants import)

2. **Module-level HUD client state:**
   ```python
   _hud_client = None
   _hud_last_reconnect = 0.0
   _HUD_RECONNECT_INTERVAL = 5.0
   _HUD_LEVEL_INTERVAL = 0.05  # 20fps throttle for audio_level messages
   _hud_last_level_send = 0.0
   ```

3. **Helper function `_hud_send(msg: dict)`:**
   - If `_hud_client` is None, return
   - Try `_hud_client.send(msg)`
   - Wrap in try/except to never crash

4. **Helper function `_hud_ensure_connected()`:**
   - If `_hud_client` is not None and `_hud_client._sock` is not None, return (already connected)
   - Check if `time.time() - _hud_last_reconnect >= _HUD_RECONNECT_INTERVAL`
   - If so, call `_hud_client.reconnect()`, update `_hud_last_reconnect`

5. **In `main()` function**, after `_kill_orphan_indicators()`:
   - Import HUDClient: `from vox.hud.ipc import HUDClient`
   - Create client: `global _hud_client; _hud_client = HUDClient(HUD_SOCKET_PATH)`
   - Call `_hud_client.connect()` (silent fail if HUD not running)

6. **In `start_recording()`**, after `_show_recording_indicator(True)`:
   - `_hud_send({"type": "state", "state": "listening"})`

7. **In `stop_recording()`**, after `_show_recording_indicator(False)`:
   - `_hud_send({"type": "state", "state": "processing"})`

8. **In `_send_local()`**, in the `finally` block (after `busy = False`):
   - `_hud_send({"type": "state", "state": "idle"})`

9. **In `_send_local()`**, after successful transcription (after the log line with transcription result):
   - `_hud_send({"type": "transcript", "text": text})`

10. **In the main loop** (inside `while not _shutdown.is_set()`), after the audio buffer append block and before silence watchdog:
    - If `_is_rec`:
      ```python
      now_level = time.time()
      if now_level - _hud_last_level_send >= _HUD_LEVEL_INTERVAL:
          _hud_last_level_send = now_level
          level = float(np.abs(audio).max()) / 32768.0  # Normalize int16 to 0.0-1.0
          _hud_send({"type": "audio_level", "level": round(level, 3)})
      ```

11. **In the main loop**, in the idle section (where health check runs), add periodic reconnect:
    - Call `_hud_ensure_connected()` inside the `if not _is_rec and not _is_busy:` block (throttled by the reconnect interval internally)

12. **In the `finally` block** of `main()`, before shutdown log:
    - `if _hud_client: _hud_client.close()`

13. **In `_cancel_recording_from_ptt()`**, after `audio_cue("paused", ...)`:
    - `_hud_send({"type": "state", "state": "idle"})`

14. **In silence watchdog cancel** (the silence timeout block), after `audio_cue("paused", cues_dir)`:
    - `_hud_send({"type": "state", "state": "idle"})`

All `_hud_send` calls must be wrapped to never throw — the HUD is optional.

  <files>vox/audio/tts.py</files>
  <action>
Add HUD IPC integration to the TTS worker so the HUD shows speaking state and queue count.

1. **Add module-level HUD client state** (after existing module-level state block):
   ```python
   _hud_client = None
   ```

2. **Add helper `_hud_send(msg: dict)`:**
   - If `_hud_client` is None, return
   - Try `_hud_client.send(msg)`
   - Except any exception: pass (never crash TTS for HUD)

3. **In `start_worker(config)`**, after the worker thread starts:
   - Import HUDClient lazily: `from vox.hud.ipc import HUDClient`
   - Import path: `from vox.constants import HUD_SOCKET_PATH`
   - Create and connect: `global _hud_client; _hud_client = HUDClient(HUD_SOCKET_PATH); _hud_client.connect()`
   - Wrap in try/except ImportError (graceful if ipc module not available)

4. **In `_tts_worker()`**, right after dequeuing an item (after `text, voice, speed = item`):
   - `_hud_send({"type": "tts_start", "text": text})`
   - `_hud_send({"type": "state", "state": "speaking"})`

5. **In `_tts_worker()`**, in the `finally` block (after `_tts_queue.task_done()`):
   - `_hud_send({"type": "tts_end"})`
   - Check if queue is empty: `if _tts_queue.empty(): _hud_send({"type": "state", "state": "idle"})`

6. **In `speak()`**, after `_tts_queue.put(...)`:
   - `_hud_send({"type": "queue_update", "count": _tts_queue.qsize()})`

7. **In `_tts_worker()`**, in the `finally` block, after `task_done`:
   - `_hud_send({"type": "queue_update", "count": _tts_queue.qsize()})`

8. **In `shutdown()`**, after the worker thread join:
   - `global _hud_client`
   - `if _hud_client: _hud_client.close(); _hud_client = None`

All HUD sends are fire-and-forget with try/except wrapping. The TTS worker must never block or crash due to HUD communication issues.
