       - `PushToTalkConfig`: enabled (bool, default True), key (str, default "fn")
       - `AudioConfig`: sample_rate (int, default 16000), chunk_size (int, default 1280)
       - `VoxConfig` (root model): wake_words (WakeWordConfig), threshold (float, default 0.5), cooldown_secs (float, default 2.0), min_recording_secs (float, default 1.5), silence_timeout_secs (float, default 5.0), silence_threshold (int, default 200), target_app (str, default "" -- EMPTY, not "Conductor"), enter_count (int, default 2), transcription_prefix (str, default ""), stt (STTConfig), tts (TTSConfig), push_to_talk (PushToTalkConfig), audio (AudioConfig), mic_priority (list[str], default ["MacBook Pro Microphone"]), cues_dir (str, default ""), log_file (str, default "/tmp/vox.log"), log_max_bytes (int, default 1_000_000)

       - If CONFIG_FILE does not exist: return VoxConfig() (all defaults)
       - Wrap pydantic ValidationError in a user-friendly print: show each error's field path and message, then sys.exit(1)
       - Accept optional `config_path: Path | None` parameter to override default location (for testing and CLI --config flag)

    4. **`generate_default_config() -> str` function:**
       - Returns a commented YAML string showing all config options with their defaults
       - Use the template from the research (lines 313-350 of RESEARCH.md) as the basis
       - Ensure target_app default is "" (empty), mic_priority default is ["MacBook Pro Microphone"], log_file is "/tmp/vox.log"
       - Include section headers as YAML comments

    5. **`ensure_config_dir()` function:**
       - Creates CONFIG_DIR if it doesn't exist
       - If CONFIG_FILE doesn't exist, writes generate_default_config() to it
       - Returns the config file path
